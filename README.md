# Gaokzx-spider

Python Spider for gaokzx.com

---

## Usage

Just run main.py and enter the link.

```shell
python main.py
```

For example:

```shell
python main.py
# 请输入目标链接(试卷汇总页面)：https://www.gaokzx.com/c/202112/57428.html
```
